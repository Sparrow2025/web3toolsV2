# web3tools

## 项目简介

> 使用React框架和MUI构建最全面的，可脱机运行的Web3工具网站


## 运行步骤

```shell
# 1. 进入my-web3-site
cd ./my-web3-site

# 2. 安装依赖
npm install

# 3. 启动项目
npm start
```
